# LavCMS

Простая CMS на Laravel для новостей и статей с админ-панелью.

## Возможности

- **Новости** и **статьи**: заголовок, краткое описание, текст, дата публикации, черновики
- **Админ-панель** (`/admin`): вход по email/паролю, CRUD для новостей и статей, дашборд
- **Публичный сайт**: главная, список и просмотр новостей и статей (только опубликованные)

## Требования

- PHP 8.2+
- Composer
- MySQL или SQLite

## Установка

1. Установите зависимости:
   ```bash
   composer install
   ```

2. Скопируйте конфигурацию и сгенерируйте ключ:
   ```bash
   copy .env.example .env
   php artisan key:generate
   ```

3. Настройте базу данных в `.env` (например, MySQL):
   ```
   DB_CONNECTION=mysql
   DB_DATABASE=lavcms
   DB_USERNAME=root
   DB_PASSWORD=
   ```

4. Выполните миграции и сидер (создаст админа и примеры контента):
   ```bash
   php artisan migrate
   php artisan db:seed
   ```

5. Запустите сервер:
   ```bash
   php artisan serve
   ```

Откройте в браузере: http://localhost:8000

## Вход в админку

- URL: http://localhost:8000/admin  
- Логин: `admin@example.com`  
- Пароль: `password`

Рекомендуется сменить пароль после первого входа.

## Маршруты

| URL | Описание |
|-----|----------|
| `/` | Главная |
| `/news` | Список новостей |
| `/news/{slug}` | Просмотр новости |
| `/articles` | Список статей |
| `/articles/{slug}` | Просмотр статьи |
| `/admin` | Админ-панель (требуется вход) |
| `/admin/login` | Вход |
